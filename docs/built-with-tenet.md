# Built With Tenet

These projects were built while testing Tenet's autonomous development loop. They are useful examples of the code, docs, `.tenet/` artifacts, critic feedback, and retry trails a Tenet run can produce:

- [tenet-example-tetrimino](https://github.com/JeiKeiLim/tenet-example-tetrimino) — A frontend-only, keyboard-first Tetris app built with Vite, React, and TypeScript. Its README records a long Tenet build journey with DAG execution, critic retries, Playwright remediation, generated Web Audio, and a local leaderboard.
- [my-home-index-server](https://github.com/JeiKeiLim/my-home-index-server) — A single-binary macOS port-manager dashboard for discovering user-owned TCP listeners in a port-forwarded range, then visiting, copying, renaming, restarting, or killing them through one authenticated URL. The repo documents a 13-hour end-to-end Tenet validation run.
- [emojink](https://github.com/JeiKeiLim/emojink) — A full-stack emoji URL shortener with public profiles, badges, reactions, trending links, interstitial redirects, and share-card rendering. The repository keeps the Tenet specs, decompositions, visuals, journals, knowledge notes, and SQLite state that produced the app.
